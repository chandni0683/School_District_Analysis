# School_District_Analysis
## Overview
The purpose of this analysis is to provide a high-level snapshot of the district's key metrics, presented in a table format, an overview of the key metrics for each school, presented in a table format. Also we will need to analyze the tables presenting each of the following metrics:
1. Top 5 and bottom 5 performing schools, based on the overall passing rate
2. The average math score received by students in each grade level at each school
3. The average reading score received by students in each grade level at each school
4. School performance based on the budget per student
5. School performance based on the school size 
6. School performance based on the type of school

## Summary
After removing the data for ninth grade for Thomas High School, the following changes were found in while performing the analysis:
1. Average scores for math, reading and overall changed
2. Total number of students changed when ninth grade data was removed for Thomas High School
3. The percentage of students who passed math, reading and overall changed
4. Scores by school type changed.
